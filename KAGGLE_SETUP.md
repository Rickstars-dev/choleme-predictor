1. **Log in to Kaggle**

4. Scroll down to **"API"** section
5. Click **"Create New API Token"**
6. A file named `kaggle.json` will download automatically

**The file looks like:**
```json
{
  "username": "your_username",
  "key": "your_secret_key_here"
```
